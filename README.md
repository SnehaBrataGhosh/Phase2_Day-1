Day 1 – Basic Input/Output Practice

Overview
This folder contains my first set of practice problems to get comfortable with Python syntax, input/output, and simple arithmetic and conditions. Each script is a small standalone solution.

Concepts Covered
- print() output
- Reading user input with input()
- Converting input using int() and float()
- Basic arithmetic (+, -, *, /, %, **)
- if/elif/else conditions
- Multiple variable assignment for swapping values

Code Highlights
- Printing a simple greeting (hello world style)
- Sum of two user-provided numbers
- A simple calculator that prints results for +, -, *, and /
- Even/odd check using modulus
- Swapping two numbers without a temporary variable
- Circle area calculation using \(r^2\) (approx pi value)
- Finding the largest of three numbers
- Showing the difference between numeric input (cast to int) and raw string input using type()
- Simple interest calculation
- Celsius to Fahrenheit conversion

Learning Outcome
I built confidence with basic Python structure: taking input, doing calculations, and writing simple branching logic. This set also helped me understand why type conversion matters when using input().
